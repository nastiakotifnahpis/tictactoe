# Features
  - Splash Screen
  - Play TicTacToe Game
  - Winner
  - Background Music
  - Light/Dark Mode
  - Help
  - Portrait/Landscape Mode
  
 # Tools/Tech/Libraries
  - Java
  - Android Studio
  - Room Persistence Library
  - LiveData
  - MVVM Architecture
  - Material Design
  - ButterKnife (Zelesny)
  - SweetAlertDialog
  - GlideToast
  - ResideMenu
